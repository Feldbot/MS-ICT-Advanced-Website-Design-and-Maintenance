// This website showcases dynamic web scripting techniques, which primarily include:

  • Javascript — the site uses scripting to dynamically inject HTML through the DOM
  • Ajax form validation — Parsley, a JS form validation library, is used on the contact form to verify fields
  • jQuery UI — the contact form fields use a widget to display dynamic tooltips
  • Flickr API — the gallery uses an API to pull images directly from Flickr servers
  • Local Storage — the credit line is served from local storage and injected through the DOM
  • Ajax GET calls — the footer IP is echoed back by httpbin.org servers
